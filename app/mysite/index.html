Дипломный проект по программе обучения на DevOps инженера в TeachMeSkills

Технологии применяемые для реализации проекта:

  • Ansible
  • Docker
  • Docker Compose
  • Docker Hub
  • Google Cloud Platform
  • Google Kubernetes Engine
  • Terraform
  • GitHub
  • Jenkins
  • Nginx
  • Certbot
  • Ingress nginx controller
  • Cert-manager kubernetes
  • Prometheus
  • Grafana

Описание реализованного проекта:

Проект состоит из двух составляющих:

  1. Подготовка, развертывание инфраструктуры.
  2. Автоматизация процессов развертывания на этапе разработки и поддержки приложения для нашего учебного проекта.
